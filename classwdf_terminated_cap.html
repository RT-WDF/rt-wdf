double wdfTerminatedCap::calculateUpRes ( double  T)
virtual

Returns the capacitors upfacing port resistance.

This function is called from adaptPorts(). It returns the upfacing port resistance of the node to fulfill termination according to the capacitors adaptation law Rup = T / ( 2.0 * C ).

Parameters
Tsample period T = 1/fs in seconds as needed to adapt the capacitor
